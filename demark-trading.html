Connect this to your Robinhood account
Robinhood Agentic Trading (beta) exposes your account to an MCP-capable agent. This tool never touches your account — it produces signals and intents; your agent places the orders after you approve.
  1. Open this tool — it's a single static file. Double-click demark-trading.html, or host it (e.g. GitHub Pages) and open the URL on any device.
  2. Connect your agent to Robinhood's MCP server at https://agent.robinhood.com/mcp/trading:
    Claude Code: claude mcp add robinhood-trading --transport http https://agent.robinhood.com/mcp/trading
    Claude Desktop: Settings → Connectors → Add custom connector → paste the URL.
  3. Authenticate & fund — Robinhood's OAuth prompt creates a separate agent sub-account and lets you move a dedicated budget into its sandboxed wallet. Your agent never sees your password and can only spend what's in that wallet.
  4. Set guardrails — choose a spending limit and turn on manual approval so you confirm each order. Every trade also appears in your Robinhood Activity; disconnect anytime in the app.
  5. Run the loop — paste Copy fetch prompt to your agent to get bars → load them here → review → paste Copy execution prompt (it embeds your approved intents) back to the agent to place the orders.
Stocks only in the current beta (options/crypto/futures “coming soon”). Verify the MCP URL on Robinhood's own Agentic Trading support page before connecting — only connect servers you trust.
How the agentic workflow fits together
  1. Fetch — Your agent (Claude + Robinhood MCP) pulls daily OHLC history for the symbol and pastes the JSON into OHLC bars.
  2. Compute — This tool runs TD Sequential locally in your browser: buy/sell Setups (9), Countdowns (13) with the bar-8 close qualifier, TDST support/resistance, and TD risk stops. No keys or data leave the page.
  3. Propose — Each actionable signal becomes an demark.order_intent.v1 object: side, limit price, sized quantity (from wallet × risk ÷ risk-per-share), stop, target, and rationale.
  4. Approve & execute — You review, then the agent submits the approved intents through Robinhood's MCP order tools. Stocks only in beta.
Intent schema
Each intent carries requiresHumanApproval:true and a risk block so the agent never sizes a position itself. Limit price defaults to the signal bar's close — adjust before sending.
Educational tool, not financial advice. DeMark Sequential is a discretionary indicator; backtest before risking capital and keep the agent wallet funded only with money you can afford to trade.
